# Per-Actor Briefing Matrix

**Audience**: SI executive committee + competitive-intel.
**Score**: 1–5 per track. *5 = clear leader with named production evidence; 4 = strong product, partial evidence; 3 = credible but partial; 2 = weak / partner-dependent; 1 = not a player in this track.*
**Tracks**: **CO** = Cognitive overlay; **TH** = Transitional hybrid; **OP** = Operational AI.
**Notes**: For vendors, readiness = "their product enables this track at production scale." For named operators / banks / insurers, readiness = "they themselves have demonstrated this track in production." Verticals: **T** = Telco, **B** = Banking, **I** = Insurance.

## Class summary — who wins which track today

| Track | Class winners (2026) | Class also-rans |
|---|---|---|
| **Cognitive overlay** | Microsoft, Google, OpenAI; Cohere; Salesforce, ServiceNow; LangGraph + LangSmith; specialist OSS-broker (HF) | All hyperscalers and frontier labs are credible here. Differentiation is increasingly *vertical context graph*, not model swap. |
| **Transitional hybrid** | Microsoft (Agent Governance Toolkit + Foundry), AWS (AgentCore Policy/Eval), Google (A2A + ADK), Anthropic (Claude SDK + MCP), Cohere (North), MS Agent Framework, LangGraph, Eclipse LMOS, Salesforce Agentforce, ServiceNow TSM, NICE Actimize, Cytora Autopilot, Shift Claims | Live battleground. The SI's defensible ground sits on top of these primitives, not at their level. |
| **Operational AI** | NVIDIA (telco), Microsoft (Commerzbank Ava is the bellwether), China Mobile (L4 trial), NTT Docomo (NOC), Lemonade (greenfield), Capital One (auto-dealer), CFC (cyber UW); Datadog/Dynatrace/LangSmith/Braintrust/Arize as the eval substrate | Most named players sit at 2–3. Few clear leaders; cancellation risk per Gartner is 40% by 2027. |

## How to use this matrix

1. **In RFP responses**, cite the rivals' overlay/transitional readiness scores when justifying why the SI's gateway-of-record is the right buy.
2. **In M&A scenario planning**, use class summaries — Cohere, Eclipse LMOS, and Cytora are the three direct-threat or partner candidates worth tracking.
3. **In CTO-office investment debates**, the operational column should slow down operational-platform capex requests; the transitional column should accelerate gateway + obligation-pack investment.
